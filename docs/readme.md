# milvus2milvus

milvus 数据导入导出工具, 支持 2.5.x 等版本

## (1) Requirements
- jdk8

## (2) Get Started

```shell

# milvus2milvus
java -cp milvus2milvus-1.0.0.jar Milvus2Milvus --uri <SRC_MILVUS_URI> --token <SRC_MILVUS_TOKEN> --collections * --t_uri <TARGET_MILVUS_URI> --t_token <TARGET_MILVUS_TOKEN>

```
